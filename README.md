# Bombando
A "Bombando" permite que tenhas, no teu bolso, os Memes mais conhecidos de Portugal!

Podes ainda definir qualquer um como:
- Toque de Chamada.
- Som de Notificação.
- Som de Alarme.

## Política de Privacidade
